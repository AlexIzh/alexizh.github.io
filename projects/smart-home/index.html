Application for SmartHomes

Jan, 2016
An iPhone/iPad application for managing the smart home system controlled by the hardware on raspbery pi.

Role: Lead iOS Developer

Team: Up to 4 developers at one time

Responsibility: Leading the development process, developing MVP and version 1.0,

Technologies: Swift, MQTT, Storyboards, CoreData, OpenSSL

Description

It’s one of the most complex applications I ever made. It contained tons of custom mechanics, animations, non-standard UI layouts. Almost every screen there contained something interesting from development side.

Just a few examples:

Demo mode: The application can work with the sample database without the Internet connection. Any user’s action should change the database as inside a real app using. For that, we implemented a simple server inside the app which was injected to our network layer just by one line. So, for the whole app there was no difference it works inside demo mode or not.

iPad version: the iPad version was a bit different from the iPhone version. Almost all table views from iPhone version converted to collection views for iPad version. For that, I implemented a simple interface which allows working with collections (table/collection/custom). You can check it out here.

DB synch and encryption: Since it’s an app for managing an entire home and a user should be able to use the app not only from home inside his local network but also outside of this, it required quite good encryption. It was done by generating a user’s certificate right in the device with encryption all messages between the device and the MQTT broker.

Also, since the home data wasn’t updated so often, we implemented full data synchronization between hardware and iOS app. A user could update any data in real-time and see changes at the same time in his iPhone/iPad device. Only data was updated very often is states of devices, getting which is way faster than loading every time every page from scratch.

Of course, there a lot of interesting things yet which I missed here, if you are interested, feel free to write an email to me.
