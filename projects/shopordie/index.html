Shop Or Die!

Mar, 2012
An iPad platformer game about traveling with a lot of mini-games and very specific graphic.

Role: Lead Game Developer

Team: The game was developed by 2 developers.

Responsibility: Leading the development process

Technologies: C++, Cocos2d-x, Box2d

Description

The game was developed on C++ because it was planned to be moved to Android as well with using Android NDK. It wasn’t done because of budget, though.

Despite the very specific design, development was very interesting and complex. It was my first game except for small Battleship or TicTacToe games, and it gave me awesome experience. The game contained a dressing room (the main character could buy and use some clothes or accessories) and a lot of mini-games. The game itself is a simple platformer with small levels but every level contained 3-4 mini-games (more 20 mini-games for the whole game) with different mechanics, so the main challenge was to implement them all.
