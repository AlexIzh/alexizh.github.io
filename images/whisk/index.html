Whisk

Jul, 2019
An iPhone application for storing and sharing recipes, creating and using shopping list and other things for cooking.

Role: iOS Developer

Team: 4 iOS developers

Responsibility: Implementing features, code review

Technologies: Swift, MVVM, Xib, Realm, RxSwift

Description

It’s one of the most good looking applications I ever made. I implemented a lot of parts there because I started to work almost from start of the project. Parts what have been made completely by me: recipe details page, recipe list (but it was updated later), share extension, item details, recipe actions and so on. It’s very interesting project from UI perspective and I was new with RxSwift, so, it was great experience for me as well.
